### 投光物
```
平行光
-- 固定光源

点光源
-- 点光源是处于世界中某一个位置的光源，它会朝着所有方向发光，但光线会随着距离逐渐衰减。

衰减
-- 随着光线传播距离的增长逐渐削减光的强度

在现实世界中，灯在近处通常会非常亮，但随着距离的增加光源的亮度一开始会下降非常快，但在远处时剩余的光强度就会下降的非常缓慢了。

Fatt=1.0/(Kc + Kl * d + Kq * pow(d,2))

聚光
-- 聚光是位于环境中某个位置的光源，它只朝一个特定方向而不是所有方向照射光线。（手电筒）

聚光的方向向量通常朝向玩家正前方

平滑/软化边缘
-- 模拟聚光有一个内圆锥(Inner Cone)和一个外圆锥(Outer Cone)。将内圆锥设置为上一部分中的那个圆锥，同时需要一个外圆锥，来让光从内圆锥逐渐减暗，直到外圆锥的边界。

-- 内圆锥之内强度为1.0，外圆锥之外强度为0.0

光强计算公式 I=（θ−γ）/ϵ
其中ϵ是内圆锥和外圆锥之间的余弦值差
```
